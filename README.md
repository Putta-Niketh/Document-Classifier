Document Classifier

Overview

This project is a simple Machine Learning Document Classifier built using Python and Scikit-Learn.

Features

- Reads text documents
- Converts text into numerical features
- Trains a Machine Learning model
- Predicts the category of new documents

Technologies Used

- Python
- Pandas
- Scikit-Learn

Dataset Categories

- Sports
- Technology
- Business

Algorithm

Multinomial Naive Bayes
